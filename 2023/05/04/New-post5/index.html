Hexo 从零开始安装教程

发表于2023-05-04|更新于2023-05-05|hexo
|阅读量:

前言

本文详细描述了如何在Github上，使用hexo部署博客

安装Hexo前提

安装node.js

安装Git

关于具体的node.js和Git的安装可以自行查阅资料，这里不做过多介绍。

安装完成后检查安装的Node.js（内置npm） 和 Git 是否安装成功

接下来准备开始安装Hexo

步骤一

找到我们的 Git Bash 并打开。

步骤二

在命令行中输入： node -v和npm -v然后回车，显示出版本号即代表我们配置环境成功啦！

步骤三

正式开始安装 Hexo ，在命令行中输入 cnpm install -g hexo-cli然后回车，静待一会等待下载完成。

步骤四

验证 Hexo 安装是否成功，在命令行中输入 hexo -v然后回车，我们即可看到关于 Hexo 的一切详细信息。

步骤五

利用命令行mkdir Blog创建我们即将使用的 Hexo 博客的目录文件夹(也可以自己手动创建)，里面将存放关于 Hexo 博客的各种配置文件、源码、文档、主题等。（如果在安装 Hexo 中出现任何问题，可以删除此目录文件夹，重新执行之前的步骤即可）。

步骤六

初始化 Hexo 博客，进入你所创建的文件夹，右键打开Git Bash在命令行中输入hexo init然后回车，等待初始化，显示INFO Start blogging with Hexo!，即为初始化成功！
